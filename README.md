# RWSS Dealership Order Form (Fall 2026)

A web-based order form that replicates the behavior of the Excel spreadsheet **Drew Lawless - FALL 2026 NEW RWSS DEALERSHIP.xlsx**.

## How to run

Open `index.html` in any modern browser (Chrome, Firefox, Safari, Edge). No server or build step required.

```bash
open index.html
# or double-click index.html in Finder
```

## What's included

- **Header** — Three sections: (1) Date, Sales Rep, P.O. #, PO Entry; (2) Account #, Store Name, Address, City, State, ZIP, Address change; (3) Terms, Our Reference.
- **Tabs** — NEW SHOES, EXISTING, EUROPEAN SHOES, CARE PRODUCTS & MISC, SOCKS, FOOTBEDS, BELTS, GLOVES & SCUFFS, QUADREX & SLIPPERS, SAFETY GLASSES, QR BELTS.
- **NEW SHOES Section 1** — Red Wing / SuperSole X: styles 2543, 3577, 3580; widths D (sizes 5–16), E2 & E3 (sizes 5–14); ship date 10/15/26.
- **Other tabs** — EXISTING, EUROPEAN, CARE PRODUCTS, SOCKS, FOOTBEDS, BELTS, GLOVES, SLIPPERS, SAFETY GLASSES, QR BELTS with product rows and section totals.
- **Grand totals** — GRAND TOTAL FOOTWEAR $ and Order Grand Total at the bottom.

## Files

- `index.html` — Structure and styles
- `app.js` — Tabs, table rendering, and all calculations
